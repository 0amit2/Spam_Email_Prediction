# Spam Email Detection

# Overview
Spam Email Detection is a Machine Learning project that classifies emails as Spam or Not Spam (Ham)
using Natural Language Processing techniques. The model is trained on labeled email data and predicts
whether a new message is spam.

# Features
- Text preprocessing and cleaning
- Feature extraction using NLP
- Machine learning classification
- Model evaluation
- Predict new emails

# Tech Stack
- Python
- Scikit-learn
- Pandas
- NumPy
- NLTK
- Matplotlib / Seaborn


# Model Workflow
1. Load dataset
2. Data cleaning
3. Text preprocessing
4. Feature extraction (CountVectorizer )
5. Train model (Naive Bayes )
6. Model evaluation
7. Prediction


# Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix
